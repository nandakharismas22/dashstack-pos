# DashStack
Website Point of Sales yang dimana menggunakan ReactJs dan Tailwind CSS di sisi Frontend dan Node.js dari sisi Backend

## Cara Running

Jalankan pada 2 terminal yang berbeda 

- Terminal 1
```bash
cd backend
nodemon index
```

- Terminal 2
```bash
cd frontend
npm install
npm start
```
